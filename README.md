# Drum Tuner
Drum Tuner - это тестовое приложение на языке Python для тестирования и отладки FFT-анализа для настройки мембран барабанов.

Приложение имеет 2 сценария использования:

 1. чтение с файла
 2. ввод с микрофона

**Чтение с файла**
В папке *src* есть тестовый файл *snare_drum.wav*, который можно запустить для простого ознакомления с приложением. Для этого проверьте, чтобы в файле *config.py* строки 3-4 были следующими:

```python
USE_FILE  =  True
FILENAME  =  os.path.join('assets', 'snare_drum.wav')
```
После этого или запустите файл main.py в вашем IDE, или откройте папку с проектом на устройстве, запустите в ней терминал и используйте запуск из терминала:
```
py main.py
```
