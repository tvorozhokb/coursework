# Drum Tuner
Drum Tuner - это тестовое приложение на языке Python для тестирования и отладки FFT-анализа для настройки мембран барабанов.

## Структура README:
```
- Структура проекта
- Требования
- Установка
- Использование
```
## Структура проекта:
```
├── assets/ 
│ └── snare_drum.wav
├── src/ 
│ ├── __init__.py 
│ ├── algorithm.py 
│ ├── config.py
│ └── gui.py
├── .gitignore 
├── README.md 
├── main.py
└── requirements.txt
```
## Требования
- Python 3.14.2
- Необходимые библиотеки

## Установка
1. Установите Python 3.14.2 с официального [сайта](https://www.python.org/downloads/release/python-3142/).
2. Клонируйте репозиторий и перейдите в папку проекта
3. Запустите в папке терминал и установите необходимые библиотеки при помощи команды:
```
pip install -r requirements.txt
```

**Важно!**
Для Linux так же необходимо вручную установить аудио-прослойку PortAudio:
```
sudo apt install libportaudio2
```
## Использование
Приложение имеет 2 сценария использования:

 1. чтение с файла
 2. ввод с микрофона

**Чтение с файла**
В папке `assets` есть тестовый файл `snare_drum.wav`, который можно запустить для простого ознакомления с приложением. Для этого проверьте, чтобы в файле `config.py` строки 3-4 были следующими:

```python
USE_FILE  =  True
FILENAME  =  os.path.join('assets', 'snare_drum.wav')
```
После этого или запустите файл `main.py` в вашем IDE, или откройте папку с проектом на устройстве, запустите в ней терминал и используйте запуск из терминала:
```
py main.py
```
Запустив приложение, вы увидите открывшееся окно:

![1scr](https://i.postimg.cc/DzHLr1bD/readme-scr1.png)

При нажатии на кнопку `PLAY` файл `snare_drum.wav` запустится в программе и вы увидите результат работы - **значение частоты**, **ближайшей ноты** и указания, если частотыа **выше**, **ниже** или **точно попадает** в ноту на экране:

![2scr](https://i.postimg.cc/q7Sys2CS/readme-scr2.png)

**Ввод с микрофона**
Для успешного ввода с микрофона вам понадобится установить в файле `config.py` следующие параметры:
```python
USE_FILE  =  False
...
DEVICE_ID  =  9
```
`DEVICE_ID` - это номер устройства, которое будет использоваться для ввода. Для того, чтобы узнать его, следует в терминале вызвать список всех аудиоустройств:
```
python -m sounddevice
```
Символами `<` и `>` будут обозначены устройства, которые использует система сейчас. Однако, возможно потребуется выбрать то же устройство из списка, но с использованием другого драйвера (например, *WASAPI* вместо *MME*).

После того, как вы установили нужные параметры в `config.py`, запустите файл `main.py` одним из ранее описанных способов.
Вас встретит похожее окно:

Как только микрофон считает звук достаточной громкости, начнётся анализ и вы увидите уже знакомый результат:
